# Upside-down

This is `vim` plugin that allows you to move your code using following shortcuts:

## Up and down
 - <kbd>alt</kbd> <kbd>↓</kbd>
 - <kbd>alt</kbd> <kbd>j</kbd>
 - <kbd>alt</kbd> <kbd>↑</kbd>
 - <kbd>alt</kbd> <kbd>k</kbd>

## Left and right
 - <kbd>alt</kbd> <kbd>←</kbd>
 - <kbd>alt</kbd> <kbd>h</kbd>
 - <kbd>alt</kbd> <kbd>→</kbd>
 - <kbd>alt</kbd> <kbd>l</kbd>
