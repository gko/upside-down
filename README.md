# Upside-down

This is `vim` plugin that allows you to move your code using following shortcuts:

## Up and down
 - <key>alt</key><key>↓</key>
 - <key>alt</key><key>j</key>
 - <key>alt</key><key>↑</key>
 - <key>alt</key><key>k</key>

## Left and right
 - <key>alt</key><key>←</key>
 - <key>alt</key><key>h</key>
 - <key>alt</key><key>→</key>
 - <key>alt</key><key>l</key>
